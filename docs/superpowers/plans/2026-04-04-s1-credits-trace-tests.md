## Task 1: Fix Python credits trace recorder

**Files:**
- Modify: `tools/retro/s1_credits_trace_recorder.py`

The recorder has three bugs:
1. **`_ready_to_start()` records start position before the demo's level has loaded** — it captures `start_x`/`start_y` from the player position of the PREVIOUS segment or the default state, not the newly loaded demo's start position.
2. **Segment metadata captures wrong zone/act** — `_start_segment()` reads zone/act immediately when `_ready_to_start()` fires, but the position in RAM at that point is the previous demo's or default.
3. **`_handle_recording()` end-of-demo detection** — uses `game_mode != GAMEMODE_DEMO` which is correct for credits flow ending, but the `redirect_level` flow path may not properly detect transitions between demos.

### Bug analysis

The `redirect_level` flow works like this:
1. ROM boots, reaches `GAMEMODE_LEVEL` (0x0C)
2. Script writes `game_mode = 0x1C` (GAMEMODE_CREDITS) to RAM
3. ROM's GM_Credits handler runs, which loads EndingDemoLoad
4. Each demo runs in `GAMEMODE_DEMO` (0x08) mode
5. Between demos, the mode briefly returns to `GAMEMODE_CREDITS` (0x1C) to show the text card

The `_ready_to_start()` detection checks `game_mode == GAMEMODE_DEMO` AND validates zone/act word matches the expected demo. This is sound. But `_start_segment()` reads `start_x`/`start_y` from RAM — and at the first frame of `GAMEMODE_DEMO`, the player position may already be at the correct starting position set by EndingDemoLoad. The real issue is that `_handle_recording()` checks `game_mode != GAMEMODE_DEMO` to stop recording, which works, but the end-of-segment transition needs to properly detect when the game mode leaves DEMO back to CREDITS.

After re-reading the code, the actual bug is simpler: **`_start_segment()` calls `self.recorder = TraceRecorder(seg_dir)` but doesn't call `self.recorder.open()`** — wait, it does call `self.recorder.open()` at line 484. Let me re-examine the actual problem.

Looking at the output data: ALL demos show `start_x=0x0050, start_y=0x03B0` and `zone=ghz, zone_id=0` — those are GHZ defaults. The recorder is reading player position at the moment `_ready_to_start()` first fires. But the LZ3 demo SHOULD have player at 0x0A00, 0x062C after EndingDemoLoad applies the lamppost state. The 0x0050/0x03B0 default means either: (a) the ready check fires before EndingDemoLoad has set the player position, or (b) the RAM read is somehow getting stale data.

The most likely cause: stable-retro `env.step()` advances one frame and returns the RAM AFTER the frame. But `_ready_to_start()` fires when game_mode first == GAMEMODE_DEMO. On that exact frame, the ROM may have just entered DEMO mode but not yet run Level_MainLoop (which sets up the player position). The Lua BizHawk version works because BizHawk Lua runs AFTER the frame completes.

**Fix:** After `_ready_to_start()` returns a valid index, wait 1 more frame before reading start position, OR read the start position from the first recorded trace frame (which is recorded after the physics tick has run).

Actually, a simpler and more robust fix: **don't read start position at `_start_segment()` time at all — read it from the first `record_frame()` call.** The trace CSV already captures the correct position. Update metadata after recording completes.

- [ ] **Step 1: Fix `_start_segment()` to defer position/zone capture**

In `s1_credits_trace_recorder.py`, modify `_start_segment()` to record a flag that the first frame hasn't been captured yet. Then in `_handle_recording()`, after the first `record_frame()` call, read the actual player position and zone from RAM and update the segment metadata.

```python
def _start_segment(self, demo_index, emu_frame):
    self.recording_segment = True
    self.current_demo_index = demo_index
    self.segment_start_emu = emu_frame
    self.first_frame_captured = False

    seg_dir = self._segment_dir(demo_index)
    self.recorder = TraceRecorder(seg_dir)
    self.recorder.reset_state()
    self.recorder.open()

    slug = DEMO_SLUGS.get(demo_index, "unknown")
    print(f"Recording credits demo {demo_index} -> {seg_dir}")

def _capture_first_frame_state(self):
    """Read actual player position/zone from RAM after first physics frame."""
    self.start_x = self.mem.u16(PLAYER_BASE + OFF_X_POS)
    self.start_y = self.mem.u16(PLAYER_BASE + OFF_Y_POS)
    self.start_zone_id = self.mem.u8(ADDR_ZONE)
    self.start_act = self.mem.u8(ADDR_ACT)
    self.start_zone_name = ZONE_NAMES.get(
        self.start_zone_id, f"unknown_{self.start_zone_id:02X}")
    self.first_frame_captured = True
    print(f"  First frame: zone={self.start_zone_name} act={self.start_act + 1} "
          f"x=0x{self.start_x:04X} y=0x{self.start_y:04X}")
    self._write_segment_metadata()
```

- [ ] **Step 2: Update `_handle_recording()` to capture first frame**

```python
def _handle_recording(self, game_mode, emu_frame):
    base_mode = game_mode & 0x7F
    if base_mode != GAMEMODE_DEMO or game_mode != GAMEMODE_DEMO:
        self._finish_segment()
        return self._capture_complete()

    if self.recorder.trace_frame >= self.args.max_trace_frames:
        raise RuntimeError(
            f"Credits demo {self.current_demo_index} exceeded "
            f"{self.args.max_trace_frames} frames")

    self.recorder.record_frame(self.mem)

    if not self.first_frame_captured:
        self._capture_first_frame_state()

    if self.recorder.trace_frame % 300 == 0:
        self._write_segment_metadata()
        print(f"  Demo {self.current_demo_index}: "
              f"{self.recorder.trace_frame} frames recorded")
    return False
```

- [ ] **Step 3: Fix `_ready_to_start()` to use wider timer window**

The existing timer check `expected_timer - 120 <= timer <= expected_timer` may be too narrow. When using `redirect_level` mode, the timer countdown starts at exactly `expected_timer` and decrements each frame. The first frame where everything is ready might have a timer value anywhere from `expected_timer` down to `expected_timer - 10` depending on how many frames pass between mode change and the check. Widen the lower bound and remove the dependency on timer for the start check — the zone/act word match + routine >= 0x02 + non-zero position is sufficient.

```python
def _ready_to_start(self, game_mode):
    if game_mode != GAMEMODE_DEMO:
        return None
    idx = self._get_active_credits_demo_index()
    if not self._should_capture(idx):
        return None

    routine = self.mem.u8(PLAYER_BASE + OFF_ROUTINE)
    expected_za = DEMO_ZONE_ACT_WORDS.get(idx, -1)
    actual_za = self.mem.u16(ADDR_ZONE)

    if (actual_za == expected_za and routine >= 0x02):
        return idx
    return None
```

## Task 4: Create AbstractCreditsDemoTraceReplayTest

**Files:**
- Create: `src/test/java/com/openggf/tests/trace/AbstractCreditsDemoTraceReplayTest.java`

This is the core of the integration. It reuses `TraceBinder`, `DivergenceReport`, `EngineDiagnostics`, and `TraceData` from the existing trace framework, but:
- Uses `DemoInputPlayer` for input instead of BK2 playback

- Runs per-frame comparison just like `AbstractTraceReplayTest`

- [ ] **Step 1: Create the abstract base class**

```java
package com.openggf.tests.trace;

import com.openggf.game.GameServices;
import com.openggf.game.ZoneFeatureProvider;
import com.openggf.game.sonic1.credits.DemoInputPlayer;
import com.openggf.game.sonic1.credits.Sonic1CreditsDemoData;
import com.openggf.level.WaterSystem;
import com.openggf.level.objects.AbstractObjectInstance;
import com.openggf.level.objects.ObjectInstance;
import com.openggf.level.objects.ObjectManager;
import com.openggf.sprites.playable.AbstractPlayableSprite;
import com.openggf.tests.HeadlessTestFixture;
import com.openggf.tests.SharedLevel;
import com.openggf.tests.rules.RequiresRomRule;
import com.openggf.tests.rules.SonicGame;
import org.junit.Assume;
import org.junit.ClassRule;
import org.junit.Test;

import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

import static org.junit.Assert.*;

/**
 * Abstract base class for Sonic 1 credits demo trace replay tests.
 *
 * <p>Like {@link AbstractTraceReplayTest} but uses ROM-internal demo input
 * via {@link DemoInputPlayer} instead of BK2 playback. Each subclass specifies
 * a credits demo index (0-7) and the corresponding trace directory.
 */
public abstract class AbstractCreditsDemoTraceReplayTest {

    @ClassRule
    public static RequiresRomRule romRule = new RequiresRomRule();

    /** Credits demo index (0-7). */
    protected abstract int creditsDemoIndex();

    /** Path to the trace directory containing metadata.json + physics.csv. */
    protected abstract Path traceDirectory();

    /** Override to supply custom tolerances. */
    protected ToleranceConfig tolerances() {
        return ToleranceConfig.DEFAULT;
    }

    protected Path reportOutputDir() {
        return Path.of("target/trace-reports");
    }

    @Test
    public void replayMatchesTrace() throws Exception {
        int idx = creditsDemoIndex();
        Path traceDir = traceDirectory();

        Assume.assumeTrue("Trace directory not found: " + traceDir,
            Files.isDirectory(traceDir));
        Assume.assumeTrue("metadata.json not found in " + traceDir,
            Files.exists(traceDir.resolve("metadata.json")));
        Assume.assumeTrue("physics.csv not found in " + traceDir,
            Files.exists(traceDir.resolve("physics.csv")));

        // Load trace data
        TraceData trace = TraceData.load(traceDir);
        TraceMetadata meta = trace.metadata();

        assertEquals("Metadata game mismatch", "s1", meta.game());

        // Load level
        int zone = Sonic1CreditsDemoData.DEMO_ZONE[idx];
        int act = Sonic1CreditsDemoData.DEMO_ACT[idx];
        SharedLevel sharedLevel = SharedLevel.load(SonicGame.SONIC_1, zone, act);
        try {
            HeadlessTestFixture fixture = HeadlessTestFixture.builder()
                .withSharedLevel(sharedLevel)
                .startPosition(
                    (short) Sonic1CreditsDemoData.START_X[idx],
                    (short) Sonic1CreditsDemoData.START_Y[idx])
                .startPositionIsCentre()
                .build();

            // Apply demo-specific state
            applyDemoState(fixture, idx);

            // Load demo input from ROM
            byte[] demoData = GameServices.rom().getRom().readBytes(
                Sonic1CreditsDemoData.DEMO_DATA_ADDR[idx],
                Sonic1CreditsDemoData.DEMO_DATA_SIZE[idx]);
            DemoInputPlayer demo = new DemoInputPlayer(demoData);

            // Run per-frame comparison
            TraceBinder binder = new TraceBinder(tolerances());
            int frameLimit = Math.min(trace.frameCount(),
                Sonic1CreditsDemoData.DEMO_TIMER[idx]);

            for (int i = 0; i < frameLimit; i++) {
                TraceFrame expected = trace.getFrame(i);

                // Advance demo input and step engine
                demo.advanceFrame();
                int mask = demo.getInputMask();
                fixture.stepFrame(
                    (mask & AbstractPlayableSprite.INPUT_UP) != 0,
                    (mask & AbstractPlayableSprite.INPUT_DOWN) != 0,
                    (mask & AbstractPlayableSprite.INPUT_LEFT) != 0,
                    (mask & AbstractPlayableSprite.INPUT_RIGHT) != 0,
                    (mask & AbstractPlayableSprite.INPUT_JUMP) != 0);

                var sprite = fixture.sprite();
                EngineDiagnostics engineDiag = captureEngineDiagnostics(sprite);

                binder.compareFrame(expected,
                    sprite.getCentreX(), sprite.getCentreY(),
                    sprite.getXSpeed(), sprite.getYSpeed(), sprite.getGSpeed(),
                    sprite.getAngle(),
                    sprite.getAir(), sprite.getRolling(),
                    sprite.getGroundMode().ordinal(),
                    engineDiag);
            }

            // Build and output report
            DivergenceReport report = binder.buildReport();

            if (report.hasErrors() || report.hasWarnings()) {
                writeReport(report, meta, idx);
            }

            System.out.println("Credits demo " + idx + ": " + report.toSummary());

            if (report.hasErrors()) {
                DivergenceGroup firstError = report.errors().get(0);
                System.err.println("\n=== Context window around first error ===");
                System.err.println(report.getContextWindow(firstError.startFrame(), 10));
                fail("Credits demo " + idx + ": " + report.toSummary());
            }
        } finally {
            sharedLevel.dispose();
        }
    }

    /**
     * Apply demo-specific state (lamppost, water, camera bounds).
     * Only LZ (demo 3) needs special setup.
     */
    private void applyDemoState(HeadlessTestFixture fixture, int idx) {
        if (idx == 3) { // LZ3 — lamppost state
            AbstractPlayableSprite player = fixture.sprite();
            player.setRingCount(Sonic1CreditsDemoData.LZ_LAMP_RINGS);

            fixture.camera().setMaxY(
                (short) Sonic1CreditsDemoData.LZ_LAMP_BOTTOM_BND);

            WaterSystem waterSystem = GameServices.water();
            int featureZone = GameServices.level().getFeatureZoneId();
            int featureAct = GameServices.level().getFeatureActId();
            waterSystem.setWaterLevelDirect(featureZone, featureAct,
                Sonic1CreditsDemoData.LZ_LAMP_WATER_HEIGHT);
            waterSystem.setWaterLevelTarget(featureZone, featureAct,
                Sonic1CreditsDemoData.LZ_LAMP_WATER_HEIGHT);

            ZoneFeatureProvider featureProvider =
                GameServices.level().getZoneFeatureProvider();
            if (featureProvider != null) {
                featureProvider.setWaterRoutine(
                    Sonic1CreditsDemoData.LZ_LAMP_WATER_ROUTINE);
            }
        }
    }

    private EngineDiagnostics captureEngineDiagnostics(AbstractPlayableSprite sprite) {
        int routine = sprite.isHurt() ? 0x04 : 0x02;
        int standOnSlot = -1;
        int standOnType = -1;
        ObjectManager om = GameServices.level() != null
                ? GameServices.level().getObjectManager() : null;
        if (om != null) {
            ObjectInstance ridingObj = om.getRidingObject(sprite);
            if (ridingObj instanceof AbstractObjectInstance aoi
                    && aoi.getSlotIndex() >= 0) {
                standOnSlot = aoi.getSlotIndex();
                standOnType = aoi.getSpawn() != null
                        ? aoi.getSpawn().objectId() : -1;
            }
        }

        int rings = sprite.getRingCount();
        int statusByte = 0;
        if (sprite.getDirection() == com.openggf.physics.Direction.LEFT)
            statusByte |= 0x01;
        if (sprite.getAir()) statusByte |= 0x02;
        if (sprite.getRolling()) statusByte |= 0x04;
        if (sprite.isOnObject()) statusByte |= 0x08;

        int camX = GameServices.camera() != null
                ? GameServices.camera().getX() : -1;

        int cursorIdx = -1, leftCursorIdx = -1, fwdCtr = -1, bwdCtr = -1;
        if (om != null) {
            int[] cursor = om.getPlacementCursorState();
            if (cursor != null) {
                cursorIdx = cursor[0];
                leftCursorIdx = cursor[1];
                fwdCtr = cursor[2];
                bwdCtr = cursor[3];
            }
        }

        int xSub = sprite.getXSubpixelRaw();
        int ySub = sprite.getYSubpixelRaw();

        return new EngineDiagnostics(routine, standOnSlot, standOnType,
                rings, statusByte, camX,
                cursorIdx, leftCursorIdx, fwdCtr, bwdCtr, "", xSub, ySub);
    }

    private void writeReport(DivergenceReport report, TraceMetadata meta, int idx) {
        try {
            Path outDir = reportOutputDir();
            Files.createDirectories(outDir);

            String prefix = "s1_credits_" + String.format("%02d", idx);
            Path jsonPath = outDir.resolve(prefix + "_report.json");
            Files.writeString(jsonPath, report.toJson());

            if (report.hasErrors()) {
                DivergenceGroup firstError = report.errors().get(0);
                Path contextPath = outDir.resolve(prefix + "_context.txt");
                Files.writeString(contextPath,
                    report.getContextWindow(firstError.startFrame(), 20));
            }
        } catch (IOException e) {
            System.err.println("Warning: failed to write report: " + e.getMessage());
        }
    }
}
```

## Notes

- **Input alignment:** Credits demos don't use BK2 files — the ROM's `DemoInputPlayer` byte-pair format is the input source. The trace CSV records the ROM's actual joypad state per frame, so input validation compares `DemoInputPlayer.getInputMask()` against `TraceFrame.input()`.
- **Lag frames:** Credits demos may have lag frames just like full-run recordings. However, since we're using `DemoInputPlayer` (not BK2), we handle lag differently — we rely on the demo timer decrement happening every frame regardless of lag. If lag frame handling proves needed, it can be added later.
- **Oscillation phase:** Credits demos likely don't need pre-trace oscillation advancement since the demo starts fresh from EndingDemoLoad (not mid-level). The metadata's `bk2_frame_offset` is 0 for credits traces.
- **Existing trace tests unaffected:** `TestS1Ghz1TraceReplay` and `TestS1Mz1TraceReplay` continue to use `AbstractTraceReplayTest` with BK2 input. The new `AbstractCreditsDemoTraceReplayTest` is a separate hierarchy.
